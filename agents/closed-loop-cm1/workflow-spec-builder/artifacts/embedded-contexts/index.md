# Embedded Contexts

At construction time, the Workflow Spec Builder appends `cm1_readme.md` (from akd-ext `akd_ext/agents/closed_loop_cm1/context/cm1_readme.md`) to its system prompt under a `## CM1 README Context` heading. This provides CM1-specific parameter semantics and namelist grounding.

The file is **not mirrored here** because it is large and its content is specific to CM1 internals; refer to the upstream akd-ext repo for the current version.
